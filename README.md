my-stamp-app（学習ログスタンプアプリ）

このアプリに込めた思い

テストの点数だけでは、生徒の頑張りは残りません。

真面目に取り組んでいても、点数に結びつかない日もある。それでも積み重ねてきたプロセスそのものを、誰にも消されない形で残したい。そんな思いからこのアプリを作りました。

先生が生徒の成長を、消えない形で祝える場所に。

できること

生徒が学習した記録を「スタンプ」として蓄積
管理者（先生）用のCRUD機能で、記録を確認・管理
シンプルなカウンター形式で、頑張りを可視化

技術構成

Flask
Firestore
Google Cloud Run

今後の展望
LangGraphとGeminiを組み合わせて、生徒ごとの弱点分析やクイズ生成に取り組んでいく予定です。
